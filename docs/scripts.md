# Scripts

## Docs list

Runs the docs lister for this repo:

```bash
npx -y tsx /Users/jfoust/.dotfiles/agents/scripts/docs-list.ts
```

## Browser tools

Start Chrome with DevTools enabled:

```bash
node /Users/jfoust/.dotfiles/agents/scripts/browser-tools.ts start
```
